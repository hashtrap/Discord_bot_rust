
# The inspirations

This project came as an idea to learn the rust programming language and better understand backend development. I also am a fan of both the shows Hazbin Hotel and Helluva Boss, so I wanted to base my bot based on those two things.

## Beforehand preparations

Before creating the project i needed to have some things already set up in order to fully test and utilise this bot.

One of those requirements is to have a Discord server where this bot can live and operate

The second thing required is to create a Discord application and select it as a bot.

## Connecting the bot to the Discord server

In order to be able to connect your bot to the server, you will need to generate a url and look it up with the browser of your choice. To be able to enable this url you will need to choose the bots functions (for ease of use administrator has been chosen for this bot).
